# ATM Interface

## Overview

The ATM Interface is a console-based Java application that simulates basic banking operations. Users can securely log in using a User ID and PIN and perform various ATM transactions.

## Features

* User Authentication
* Deposit Money
* Withdraw Money
* Transfer Funds
* Transaction History
* Balance Inquiry
* Logout Functionality

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Scanner Class

## How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA or any Java IDE.
3. Compile and run `ATMInterface.java`.
4. Login using the configured credentials.

## Learning Outcomes

* Java Programming Fundamentals
* OOP Concepts
* User Authentication
* Transaction Management
