Number Guessing Game
Number Guessing Game
Overview

The Number Guessing Game is a simple Java console-based application where the computer generates a random number, and the player attempts to guess it within a limited number of attempts. The game provides hints indicating whether the guessed number is too high or too low until the correct number is found.

Features
Random Number Generation
User Input Validation
Limited Guess Attempts
Hint System (Higher/Lower)
Score Tracking
Multiple Rounds Support
Console-Based Interface
Technologies Used
Java
Object-Oriented Programming (OOP)
Random Class
Scanner Class
How to Run
Clone the repository.
Open the project in IntelliJ IDEA or any Java IDE.
Compile and run NumberGuessingGame.java.
Enter your guesses and follow the hints to find the correct number.
Game Rules
The system generates a random number.
The player attempts to guess the number.
The system provides feedback whether the guess is too high or too low.
The game continues until the correct number is guessed or attempts are exhausted.
The final score is displayed at the end of the game.
Learning Outcomes
Java Programming Fundamentals
Random Number Generation
Conditional Statements and Loops
User Input Handling
Game Logic Development

Bouthya Battu
